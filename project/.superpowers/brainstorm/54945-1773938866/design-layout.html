Layout Structure

  • Sidebar (240px): Brand + Add Task button (with Q shortcut), primary GTD nav with badges, review section, manage section, user footer
  • Main content (flexible): Serif page title + subtitle, task list with inline add-task, content adapts per page
  • Side panel (340px, conditional): Slides open when a task is clicked. Shows all task metadata fields. Closes with ✕.

Design Identity

  • BW color palette: Türkis primary, warm grays, yellow accents
  • Serif titles (Georgia) + sans-serif body (system font)
  • Minimal and flat — no cards, no heavy shadows, thin dividers
  • Active items use türkis-surface highlight
